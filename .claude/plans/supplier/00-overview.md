# Supplier Module — Overview

## Purpose
Manage supplier records. Suppliers are the companies we purchase inventory from.
Every Purchase Order must belong to a supplier.

## Supplier Code
Auto-generated by system on creation: `SUP-0001`, `SUP-0002` etc.
Sequential, never resets, never editable after creation.

## Features
| # | Feature |
|---|---------|
| 1 | List suppliers — paginated, search by name/code, filter by active status |
| 2 | View supplier — detail page with linked PO history |
| 3 | Create supplier — name (unique), code (auto-generated), contact details |
| 4 | Edit supplier — update all fields except code |
| 5 | Deactivate supplier — soft delete, blocks new POs from being created for this supplier |
| 6 | Restore supplier |
