# Supplier Module — Model & Factory

## Model: `Supplier`

```php
<?php
// app/Models/Supplier.php

declare(strict_types=1);

namespace App\Models;

use Illuminate\Database\Eloquent\Factories\HasFactory;
use Illuminate\Database\Eloquent\Model;
use Illuminate\Database\Eloquent\Relations\HasMany;
use Illuminate\Database\Eloquent\SoftDeletes;
use Spatie\Activitylog\LogOptions;
use Spatie\Activitylog\Traits\LogsActivity;

class Supplier extends Model
{
    use HasFactory, LogsActivity, SoftDeletes;

    protected $fillable = [
        'name',
        'contact_name',
        'contact_email',
        'contact_phone',
        'address',
        'notes',
        // code intentionally excluded — auto-generated by SupplierService::generateCode(), never user-editable
        // is_active intentionally excluded — set only via SupplierService::deactivate()/restore()
    ];

    protected function casts(): array
    {
        return [
            'is_active' => 'boolean',
        ];
    }

    public function getActivitylogOptions(): LogOptions
    {
        return LogOptions::defaults()->logFillable()->logOnlyDirty();
    }

    // ── Scopes ───────────────────────────────────────────────────────────────────

    public function scopeActive($query): \Illuminate\Database\Eloquent\Builder
    {
        return $query->where('is_active', true);
    }

    // ── Relations ────────────────────────────────────────────────────────────────

    // UNCOMMENT after PurchaseOrder module is implemented:
    // public function purchaseOrders(): HasMany
    // {
    //     return $this->hasMany(PurchaseOrder::class);
    // }
}
```

## Factory: `SupplierFactory`

```php
<?php
// database/factories/SupplierFactory.php

declare(strict_types=1);

namespace Database\Factories;

use App\Models\Supplier;
use Illuminate\Database\Eloquent\Factories\Factory;

class SupplierFactory extends Factory
{
    protected $model = Supplier::class;

    public function definition(): array
    {
        return [
            'code'          => $this->faker->unique()->numerify('SUP-####'),
            'name'          => $this->faker->unique()->company(),
            'contact_name'  => $this->faker->name(),
            'contact_email' => $this->faker->companyEmail(),
            'contact_phone' => $this->faker->phoneNumber(),
            'address'       => $this->faker->address(),
            'notes'         => null,
            'is_active'     => true,
        ];
    }

    public function inactive(): static
    {
        return $this->state(['is_active' => false, 'deleted_at' => now()]);
    }
}
```

---

## Notes

- `code` is set by `SupplierService::generateCode()` before `create()`. Factory uses `fake()->unique()->numerify()` for test uniqueness — format matches but values are not sequential. Tests must never assert specific code values (e.g. `SUP-0001`) unless testing `generateCode()` directly in the service unit test.
- `LogsActivity` records all changes to fillable fields. Spatie activity log is the audit trail.
- `SoftDeletes` — `deleted_at` is the deactivation marker. Never hard-delete.
- `scopeActive()` — used by PO creation dropdowns to filter available suppliers.
